## Tiers

| Tier | Price | What you get |
|------|-------|--------------|
| Backer | $5/mo | Sponsor badge + name in this file |
| Individual supporter | $10/mo | Above + thank-you in next release notes |
| Logo sponsor | $25/mo | Logo or name in the project README |
| Team supporter | $100/mo | Priority bug & feature triage (best-effort, 3 business days) |
| Sustaining sponsor | $500/mo | Prominent README placement + monthly 30-min office hour |
| Embedded support | $1,000/mo | I join your team's chat for DreamGraph questions |

One-time tiers (release-notes mention, pair programming, consulting, sponsored
bugfixes, contract work) are listed on the
[sponsors page](https://github.com/sponsors/mmethodz).

## How sponsors are added here

Sponsors are added to this file manually after each new sponsorship.
If you've sponsored and don't see yourself listed within a week, please open
an issue (or DM) and I'll fix it.

If you'd prefer **not** to be listed publicly, just say so when you sponsor —
your support is appreciated either way.
