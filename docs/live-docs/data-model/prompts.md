# Prompts

> Defines the prompts used for cognitive tasks within the DreamGraph system. Prompts are essential for guiding the cognitive engine in generating responses or making decisions.

**Table:** `prompts`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| id | string | Unique identifier for the prompt. |
| content | string | The text of the prompt used in cognitive tasks. |
