# Policy Profile

> Defines a set of policies that govern the behavior of instances within the DreamGraph system. Policy profiles are crucial for managing compliance and operational guidelines.

**Table:** `policy_profile`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| id | string | Unique identifier for the policy profile. |
| policies | array | List of policies included in the profile. |
