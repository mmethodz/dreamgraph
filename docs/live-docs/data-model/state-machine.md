# State Machine

> Defines the state transitions and behaviors for instances within the DreamGraph system. State machines are essential for managing complex workflows and interactions.

**Table:** `state_machine`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| id | string | Unique identifier for the state machine. |
| transitions | array | List of state transitions defined in the state machine. |
