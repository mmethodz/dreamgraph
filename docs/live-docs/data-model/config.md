# Configuration

> Holds the configuration settings for the DreamGraph application, including transport modes and port settings. It is used to initialize the server and manage runtime parameters.

**Table:** `config`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| transport | string | The transport mode for the server, either 'stdio' or 'http'. |
| port | number | The port number for HTTP transport mode. |
