# Graph Enrichment

> Represents the process of enhancing the knowledge graph with additional metadata, labels, and links. This model is essential for improving the cognitive capabilities of the DreamGraph system.

**Table:** `graph_enrichment`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| domainMap | object | Maps entity IDs to their respective domain labels. |
| featureLinks | array | Cross-references between entities for enrichment. |
