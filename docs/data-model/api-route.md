# API Route

> Defines the HTTP endpoints available in the DreamGraph application. Each route corresponds to a specific functionality and is accessible via REST.

**Table:** `api_route`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| method | string | The HTTP method used for the route, such as GET or POST. |
| path | string | The path of the API endpoint. |
