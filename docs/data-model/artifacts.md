# Artifacts

> Stores various artifacts related to the DreamGraph project, including documentation and generated files. This data is useful for tracking project outputs.

**Table:** `artifacts`  
**Storage:** file-system  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| artifact_id | string | Unique identifier for the artifact. |
| location | string | File system path where the artifact is stored. |
