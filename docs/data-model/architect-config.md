# Architect Configuration

> Holds configuration settings for the Architect LLM provider, including the model and API key. This configuration is essential for integrating with external LLM services.

**Table:** `architect_config`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| provider | string | The LLM provider being used, such as OpenAI or Anthropic. |
| apiKey | string | The API key for authenticating with the LLM provider. |
