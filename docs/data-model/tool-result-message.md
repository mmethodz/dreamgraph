# Tool Result Message

> Represents the result of a tool call, including any output generated by the tool. This message format is used to relay results back into the conversation context.

**Table:** `tool_result_message`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| role | string | Role of the message sender (user or assistant). |
| content | array | Array of results from the tool call. |
