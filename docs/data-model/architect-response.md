# Architect Response

> Represents the response from the LLM after processing a request, including content and token usage metrics. This structure is essential for understanding the output from the LLM and managing interactions.

**Table:** `architect_response`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| content | string | The content returned by the LLM. |
| durationMs | number | Time taken to generate the response in milliseconds. |
