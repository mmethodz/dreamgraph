# Policy Profile

> Defines a set of policies that govern the behavior of the DreamGraph application. Profiles can be switched dynamically based on application needs.

**Table:** `policy_profile`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| name | string | The name of the policy profile. |
| rules | array | The rules defined within the policy profile. |
