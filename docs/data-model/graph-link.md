# Graph Link

> Represents a relationship between two entities in the knowledge graph, including metadata about the relationship. It is used to enhance the connectivity and context of the graph.

**Table:** `graph_link`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| target | string | The ID of the target entity in the graph. |
| type | string | The type of the link, such as feature or workflow. |
