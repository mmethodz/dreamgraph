# DreamGraph Documentation

This directory contains DreamGraph documentation, but the **canonical living docs entry** is:

- [docs/index.md](index.md)

`index.md` is generated from the current DreamGraph knowledge graph and should be treated as the authoritative documentation landing page for:

- features
- workflows
- data model
- architecture decisions
- UI registry
- API reference
- cognitive status

## Manual / narrative documents

Some files in this directory are hand-written narrative or reference documents. They may provide additional background, but they are not the authoritative inventory of the current system state unless explicitly regenerated.

For the current graph-grounded documentation, start here:
