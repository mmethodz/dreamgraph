# Architect Configuration

> Holds configuration settings for the Architect LLM provider, including the model, API key, and endpoint. This configuration is essential for integrating LLM capabilities into the DreamGraph system.

**Table:** `architect_config`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| provider | string | The LLM provider being used (e.g., OpenAI, Anthropic). |
| apiKey | string | API key for authenticating with the LLM provider. |
