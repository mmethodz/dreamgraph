# API Surface

> Defines the API endpoints available in the DreamGraph application. It includes methods for interacting with the application.

**Table:** `api_surface`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| method | string | HTTP method for the endpoint (GET, POST, etc.). |
| path | string | Path of the API endpoint. |
| description | string | Description of what the endpoint does. |
