# Tool Proxy

> Acts as an intermediary for tool calls within the DreamGraph application. It manages communication between the application and external tools.

**Table:** `tool_proxy`  
**Storage:** memory  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| toolId | string | Unique identifier for the tool. |
| status | string | Current status of the tool proxy. |
