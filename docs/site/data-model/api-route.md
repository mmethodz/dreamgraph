# API Route

> Defines the REST API endpoints available in the DreamGraph application. It includes methods for interacting with the instance and graph context.

**Table:** `api_route`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| method | string | HTTP method for the route (GET, POST, etc.). |
| path | string | Path of the API endpoint. |
