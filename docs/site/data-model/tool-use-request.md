# Tool Use Request

> Defines a request to use a tool within the LLM context, including the tool's name and input parameters. This structure is used to facilitate interactions with external tools during LLM processing.

**Table:** `tool_use_request`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| id | string | Unique identifier for the tool use request. |
| name | string | Name of the tool being requested. |
