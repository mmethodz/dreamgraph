# Policy Profile

> Defines a set of policies that govern the behavior of instances within the DreamGraph system. Policy profiles are used to manage compliance and operational constraints.

**Table:** `policy_profile`  
**Storage:** memory  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| id | string | Unique identifier for the policy profile. |
| name | string | Human-readable name of the policy profile. |
