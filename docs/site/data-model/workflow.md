# Workflow

> Describes a sequence of steps or actions that need to be performed within the DreamGraph system. Workflows are used to automate processes and enhance the cognitive capabilities of the system.

**Table:** `workflow`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| id | string | Unique identifier for the workflow. |
| trigger | string | Event that triggers the workflow. |
