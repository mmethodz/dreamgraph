# Graph Link

> Represents a connection between two entities in the knowledge graph, including metadata about the relationship. Graph links are crucial for navigating and understanding the relationships within the graph.

**Table:** `graph_link`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| target | string | ID of the target entity. |
| type | string | Type of the link (feature, workflow, etc.). |
