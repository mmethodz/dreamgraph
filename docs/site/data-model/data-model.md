# Data Model

> Defines the structure of data entities used within the DreamGraph application. It includes field definitions and relationships between entities.

**Table:** `data_model`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| entityName | string | Name of the data entity. |
| fields | array | List of fields that define the entity. |
