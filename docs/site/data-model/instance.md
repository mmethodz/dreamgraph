# Instance

> Represents a single instance of the DreamGraph application, including its configuration, state, and policies. It is crucial for managing multiple environments and ensuring isolation between them.

**Table:** `instance`  
**Storage:** memory  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| id | string | Unique identifier for the instance. |
| status | string | Current status of the instance. |
