# API Contract

> Defines the structure and expected behavior of the API endpoints in the DreamGraph application. This ensures that the frontend and backend communicate effectively.

**Table:** `api_contract`  
**Storage:** json  

## Fields

| Field | Type | Description |
|-------|------|-------------|
| method | string | HTTP method for the API endpoint (GET, POST, etc.). |
| path | string | Endpoint path for the API. |
