# ADR-066: Webhook-based Integration with External Services

**Date:** 2026-04-25T11:08:30.168Z  
**Status:** accepted  
**Decided by:** system  
**Affected entities:** dreamgraph_src_tools  

## Context

Need for asynchronous communication with external services.

**Constraints:**
- Integration must be non-blocking
- Must handle retries for failed requests

## Decision

Webhook-based integration

### Alternatives Considered

- **Polling** — Rejected: Inefficient and increases load on services
- **Direct API calls** — Rejected: Synchronous and blocks processing

## Consequences

- Improved responsiveness of the application
- Reduced load on external services

**Risks accepted:**
- Webhook failures could lead to data inconsistencies

## Guard Rails

> ⚠️ Do NOT change integration pattern without reviewing this ADR
