# ADR-030: Shared DataGrid Component for All Data Tables

**Date:** 2026-04-18T23:11:40.783Z  
**Status:** accepted  
**Decided by:** system  
**Affected entities:** feature_ui_registry, data_model_ui_element  

## Context

To ensure consistency and reduce duplication in UI components.

**Constraints:**
- All data tables must use the shared DataGrid component
- Must support various data formats

## Decision

A shared DataGrid component is used for all data tables.

### Alternatives Considered

- **Custom components for each table** — Rejected: Increased maintenance burden.
- **Third-party libraries** — Rejected: Dependency on external updates.

## Consequences

- Consistent UI and reduced development time.

**Risks accepted:**
- Single point of failure if the component has bugs.

## Guard Rails

> ⚠️ Do NOT change the DataGrid implementation without reviewing this ADR
