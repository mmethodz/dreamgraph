# ADR-064: API Key Authentication for Internal Routes

**Date:** 2026-04-25T11:08:30.129Z  
**Status:** accepted  
**Decided by:** system  
**Affected entities:** dreamgraph_src_api  

## Context

Need to secure internal APIs from unauthorized access.

**Constraints:**
- All internal API routes must require authentication
- API keys must be stored securely

## Decision

API key authentication

### Alternatives Considered

- **OAuth** — Rejected: Overhead for internal services
- **No authentication** — Rejected: Insecure and exposes sensitive data

## Consequences

- Increased security for internal APIs

**Risks accepted:**
- API key leakage could lead to unauthorized access

## Guard Rails

> ⚠️ Do NOT change authentication mechanism without reviewing this ADR
