# ADR-063: Use of JSON for Configuration

**Date:** 2026-04-25T11:08:30.109Z  
**Status:** accepted  
**Decided by:** system  
**Affected entities:** dreamgraph_src_config  

## Context

Configuration management needs to be simple and easily editable.

**Constraints:**
- Configuration files must be human-readable
- No complex parsing libraries should be required

## Decision

JSON files for configuration management

### Alternatives Considered

- **YAML** — Rejected: Requires additional parsing libraries
- **XML** — Rejected: More verbose and less readable

## Consequences

- Easier configuration management
- Faster onboarding for new developers

**Risks accepted:**
- Potential for syntax errors in JSON files

## Guard Rails

> ⚠️ Do NOT change the configuration format without reviewing this ADR
