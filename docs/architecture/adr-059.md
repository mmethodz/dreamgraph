# ADR-059: Shared DataGrid Component for All Data Tables

**Date:** 2026-04-25T11:03:26.924Z  
**Status:** accepted  
**Decided by:** system  
**Affected entities:** data_model_ui_element  

## Context

Need for consistency in UI presentation across different data tables.

**Constraints:**
- All data tables must use a shared DataGrid component.

## Decision

Shared DataGrid component for consistency.

### Alternatives Considered

- **Multiple Custom Components** — Rejected: Increases maintenance burden.
- **Third-Party Libraries** — Rejected: Dependency on external updates.

## Consequences

- Consistent user experience and easier updates.

**Risks accepted:**
- Single point of failure if the DataGrid component has issues.

## Guard Rails

> ⚠️ Do NOT change the DataGrid implementation without reviewing this ADR.
